- このFigmaファイルは、デザイナーや開発者のためのサポート資料や参考資料となることを目的としており、Material Theme BuilderとMaterial 3 Design Kitを使用している
- Android Developersブログでは、AndroidにおけるNowのデザインの旅について詳しく説明している
