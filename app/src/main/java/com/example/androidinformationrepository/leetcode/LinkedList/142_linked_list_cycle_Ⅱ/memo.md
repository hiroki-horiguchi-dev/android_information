# アルゴリズムについて
- fast: 2つ先の要素
- slow: 1つ先の要素
- 141 と同様にこれでサイクルがあることが判明する
- もし循環リストなのであれば、slow の位置をノードの先頭に変更し、fast の速度を変更する
- slow: 先頭のノードに戻す
- fast: 1つ先の要素をみるように変更する
- んで回して、slow fast が出会った時、そこがサイクルの始点ノードである
